PORTFOLIO

  • Corporate Websites 'RECORD-b',

  • a Concert Hall Concept.

1200px을 기준으로 작업된 뮤직 콘서트홀 컨셉의 기업형 웹사이트 'RECORD-b'입니다. 실제로 기업에서 사용할 목적을 포커싱에 두고 만들었기 때문에, 과한 인터렉션은 자제하면서도 깔끔하고 힙한 분위기를 만드려고 노력하였습니다. 사이트는 반응형에 Main / ABOUT / YOUTUBE / GALLERY / COMMUNITY / LOCATION / 회원가입 페이지로 구성되어 있으며 다소 어두운 배경에 스포트라이트를 받는 듯한 컨셉으로 밝은 포인트를 두었습니다. 기능 부분에서는 YOUTUBE와 GALLERY 페이지에서 youtube API와 flickr API를 사용하여 동적으로 컨텐츠 정보를 불러오도록 적용시켰으며, LOCATION 페이지에서는 카카오 맵을 연동한 위치 서비스를 구현하였습니다.

jQuery로만 구현한 기존의 기업형 웹사이트와 더불어, 바닐라스크립트로 변형하여 작업한 버전의 웹사이트도 준비하였습니다. 기존의 사이트와 컨텐츠와 기능면에서는 거의 동일하지만, 디자인과 컬러는 더 많은 면을 보여드리기 위해 조금의 차별점을 두어 제작하였습니다. 해당 사이트는 밝고 활동적인 컬러를 포인트로 사용 하였습니다.

PORTFOLIO

  • Isotope Grid Design site 'ounces',

  • and React Portfolio 'Saturday'.

그리드 디자인에 isotope 플러그인을 결합하여 제작한 칵테일바 컨셉의 웹사이트 'ounces'입니다. 원페이지 반응형 구조로 만들어졌기에 웹브라우저의 폭에 따라 자연스럽게 변화하는 사이트의 모습을 볼 수 있습니다. 중간중간 재미있는 기믹들도 포함시켜서 보는이들로 하여금 흥미를 가지고 브랜드를 알아갈 수 있도록 제작하였습니다. 컬러는 화려한 칵테일의 색감을 돋보이게 하기 위해 채도가 낮은 색을 배경으로 채택하였습니다.

다음으로 소개드릴 포트폴리오는 React를 통해 제작한 여행사 컨셉 웹사이트 'Saturday'입니다. 현재 React를 공부하며 구조에 대해 알아가는 과정에 있어서, 능숙하진 않지만 공부하고 있는 리액트 구조 사용에 대한 경험치를 보여드리기 위해 실제로 현장에서 사용하는 것보다 과하게 컴포넌트를 분할하여 제작하였습니다. 기능은 기업형 웹사이트 'RECORD-b'와 같이 flickr API를 사용하였습니다.
