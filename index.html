1200px을 기준으로 반응형 사이트로 작업된 뮤직 콘서트홀 기업형 웹사이트 'D-RECORD'입니다. 사이트는 메인페이지 / ABOUT / YOUTUBE / GALLERY / COMMUNITY / LOCATION / 회원가입 페이지로 구성되어 있으며, YOUTUBE와 GALLERY 페이지는 flickr API를 이용하여 동적으로 이미지와 재생목록을 불러오도록 적용시켰으며 LOCATION 페이지에서는 카카오 맵을 연동한 위치 서비스을 구현하였습니다.
